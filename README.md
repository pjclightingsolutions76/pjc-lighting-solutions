# PJC Lighting Solutions Website

This is a complete Next.js website prepared for deployment on Vercel.

## Upload to GitHub

1. Unzip this project.
2. Open your new GitHub repository.
3. Click **Add file → Upload files**.
4. Upload the contents of this folder, not the outer ZIP file.
5. Commit the files.

Your repository root should show:

- `app`
- `public`
- `package.json`
- `README.md`

## Deploy on Vercel

1. Sign in to Vercel with GitHub.
2. Click **Add New → Project**.
3. Import the GitHub repository.
4. Leave Framework Preset as **Next.js**.
5. Leave Root Directory and Output Directory unchanged.
6. Click **Deploy**.
